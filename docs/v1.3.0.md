## Changes

### SEO вҖ” `frontend/index.html`

- **Page title** updated from `Selkokielelle` to `Selkokielelle вҖ“ Muunna teksti selkokielelle`
- **Meta description** added: `Ilmainen tyГ¶kalu tekstin muuntamiseen selkokielelle tekoГӨlyn avulla.`
- **Canonical tag** added: `<link rel="canonical" href="https://selkokielelle.online/" />`

### Tagline вҖ” `frontend/index.html`

- Changed from `Muuta teksti selkosuomeksi.` вҶ’ `Muunna teksti selkokielelle.`
  - Aligns with the button label and the service name

### Copy button вҖ” `frontend/index.html`

- Replaced bare `navigator.clipboard.writeText()` call with a robust two-path implementation:
  - Primary: `navigator.clipboard.writeText()` (modern browsers)
  - Fallback: `execCommandFallback()` using `document.execCommand('copy')` (iOS Safari, older WebViews)
- Added explicit failure state: button shows `вң• Kopiointi epГӨonnistui` if both paths fail
- Fixes copy button on mobile (iOS Safari) where `navigator.clipboard` is unavailable without HTTPS focus

### tietosuoja.html вҖ” two fixes

- **Button label reference:** `"Muunna selkokielelle"` вҶ’ `"Muunna"` to match the actual button label
- **Logging disclosure:** Updated privacy text to accurately reflect that the server logs standard technical data (request timestamps, IP addresses) but not text content. Previous wording incorrectly stated nothing is logged at all.

### Backend вҖ” `backend/main.py`

- **HTTP-Referer / X-Title headers** corrected from `.fi` to `.online` domain:
  - `"HTTP-Referer": "https://selkokielelle.online"`
  - `"X-Title": "selkokielelle.online"`
