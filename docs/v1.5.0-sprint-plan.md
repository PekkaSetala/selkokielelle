# v1.5.0 Sprint Plan — selkokielelle.fi

**Version**: 1.5.0 (not 1.4.x — this release adds new capabilities, not just patches)
**Rationale**: Semver MINOR bump because this release adds security headers, accessibility landmarks, focus management, new error handling UX, output sanitization, and deploy improvements. These are user-facing behavioral changes, not just bug fixes.

**Goal**: Solid, accessible, safe, ready for real users.

**Extension stays at 1.0.0** in `manifest.json` — it has its own versioning for Chrome Web Store.

---

Each sprint contains exactly 10 blocks. Each block is a self-contained task with:
- **Files** to modify (exact paths)
- **What to do** (exact code changes, line references to current code)
- **How to verify** (test steps)

Implement blocks in order within each sprint. Commit after each block. Tag `v1.5.0` after Sprint 3.

```javascript
      const div = document.createElement('div');
      if (err.name === 'AbortError') {
        div.textContent = 'Tämä kestää odotettua kauemmin. Yritä uudelleen tai kokeile lyhyempää tekstiä.';
      } else {
        div.textContent = 'Yhteysongelma. Tarkista verkkoyhteys ja yritä uudelleen.';
      }
      outputEl.innerHTML = `<p>${div.innerHTML}</p>`;
      outputWrap.className = 'output state-done';
      outputEl.classList.add('visible');
    } finally {
      btnConv.classList.remove('is-loading');
      btnConv.removeAttribute('aria-label');
      btnConv.removeAttribute('aria-busy');
      btnConv.disabled = inputEl.value.length === 0 || inputEl.value.length > MAX;
    }
```
